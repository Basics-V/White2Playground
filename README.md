# White2Playground
"*When you don't have anywhere else to put it...*"

> [!WARNING]
> **White2Playground is not a user-friendly project.  
> Basic programming knowledge and experience is *required*.**

## Features
- [Phenomenon spawn rate control](## "PHENOM_RAND_MAN")
- [Skip intro movie](## "SKIP_INTRO_MOVIE")
- [Disable save button in the FieldMenu](## "NO_SAVE")
- [Skip intro dialogue (with forced default name/gender)](## "SKIP_INTRO_DIALOGUE")
- [Pokeball toss animation can include NPC origins](## "BALLTOSS_ORIGIN_MAN")

## Compiling
Pick your poison:
```sh
$ make code debug=desmume
$ make code debug=melonds mods="PHENOM_RAND_MAN SKIP_INTRO_MOVIE"
$ make esdb
$ make
```

## How to use
Open CTRMap, install PMC, and use the final `.yml` and `.dll` from this project.
